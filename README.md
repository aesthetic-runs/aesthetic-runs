# Aesthetic Runs

Runs that are easy on the eyes.

## Start development

### To run front end locally
`nvm use lts/hydrogen`
`cd front-end`
`npm install`
- Add API key to `/src/components/MapDisplay`
`npm start`

### To run back end locally
Ensure you are in the root folder
`nvm use lts/hydrogen`
`npm install`
`
